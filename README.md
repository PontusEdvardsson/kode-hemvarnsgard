# Kode Hemvärnsgård

Statisk GitHub Pages-sida för Kode Hemvärnsgård.

## Filer

- `index.html` innehåller sidans innehåll.
- `styles.css` innehåller layout, färger och responsiv design.
- `assets/` innehåller underlagsbilderna som användes för första versionen.

## Publicering

Tänkt repo-namn: `kode-hemvarnsgard`.

När repot finns på GitHub:

```powershell
git remote add origin https://github.com/PontusEdvardsson/kode-hemvarnsgard.git
git branch -M main
git push -u origin main
```

Aktivera sedan GitHub Pages i repots inställningar: `Settings` -> `Pages` -> `Deploy from a branch` -> `main` och `/root`.
